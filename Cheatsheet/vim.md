# Switch mode

- Insert mode: press `I`
- Command mode: press `ESC`
- Visual: Shift + v (to select character)

# Action

- ✂️  Delete current line : dd
-  Copy selected character : y
-  📰  Paste: p (after cusor) / P (before cursor)
- Copy whole file: y+
- Go to line 42: 42G
-  👈  Undo: u
-  👉  Redo: Ctrl + r
- 🔬  Search:
  - press `/`
  - type pattern
  - press `Enter`
  - press n/N to go next/previous result

# Command

- Quit :q
- Force quit :q!
- Write and quit :wq
